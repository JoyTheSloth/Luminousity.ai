# Luminousity.ai

A retro-futuristic, "alive" AI assistant that helps you navigate life based on your emotional state.

## Features
- **Mood Calibration**: Interactive mood selector to gauge your state.
- **Alive Chat Interface**: Terminal-style chat with simulated AI reasoning and "breathing" animations.
- **Cassette Futurism UI**: Dark mode, neon glows, CRT scanlines, and tactical grid background.

## Tech Stack
- React
- Vite
- Vanilla CSS (Custom Design System)

## Setup
1. `npm install`
2. `npm run dev`
